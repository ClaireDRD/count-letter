# Count letters

## Description
Write a function countLetters that **takes a string returns an object** where letters are keys and occurences of those letters are values.

Example:
```javascript
countLetters('hello')

// returns:
{
    'h': 1,
    'e': 1,
    'l': 2,
    'o': 1,
}
```

## Install
```
npm install
```

## Requirements
- Pass all the tests
- To run the tests: `npm test`
